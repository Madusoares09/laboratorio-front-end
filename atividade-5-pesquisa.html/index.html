O elemento HTML < sub > define o texto subscrito.

O texto subscrito aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor.

O texto subscrito pode ser usado para fórmulas químicas, como H 2 0, ou para escrever palavras em baixo

Exemplo simples de uso

Um exemplo simples é: O elemento H < sub > 2 < /sub > 0 é água. Neste exemplo o "2" será exibido como subscrito abaixo da linha de base da palavra "H"
